# Visualizador 3D y AR

Este proyecto es un visualizador 3D y AR para archivos .glb alojados en la carpeta `Modelos`.

## Cómo usar

1. Coloca tus archivos .glb en la carpeta `modelo`.
2. Actualiza `models.json` con la lista de tus modelos.
3. Sube el repositorio a GitHub.
4. Ve a Settings > Pages y selecciona la rama principal como source.
5. El sitio estará disponible en `https://tuusuario.github.io/turepo`.

## Características

- Visualización 3D normal con Three.js.
- Modo AR usando WebXR (compatible con dispositivos que lo soporten).

## Dependencias

- Three.js
- GLTFLoader
