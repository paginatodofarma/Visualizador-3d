# Visualizador 3D + AR (.glb) para GitHub Pages

## Archivos
- `index.html`: página principal con model-viewer.
- `model.glb`: tu modelo 3D (debes subirlo tú mismo al repositorio).

## Uso
1. Copia `model.glb` en la misma carpeta del `index.html`.
2. Haz commit y push a tu repositorio GitHub.
3. En configuración de GitHub, habilita `GitHub Pages` con la rama `main` y carpeta `/`.
4. Abre `https://<tu_usuario>.github.io/<tu_repo>/index.html`.

## Notas
- AR funciona en dispositivos soportados (WebXR, Scene Viewer, Quick Look).
- Verifica que tu `.glb` tenga texturas embebidas o rutas correctas.
